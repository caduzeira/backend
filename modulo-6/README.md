# Procedural & Orientada a objetos
- O método procedural utiliza apenas um único objeto
- Já a orientada é dividida em vários objetos, um para cada função.

## Algoritmo em portugol
1. Abrir o olho
2. Levantar da cama
3. Se estiver muito confortavel;
    1. Pegar a coberta
    2. Voltar a deitar
    3. Se cobrir
4. Se tiver que trabalhad;
    1. Levantar da cama
    2. Ficar com preguiça
    3. Ir assim mesmo
    4. Sentar na cadeira
    5. Começar a trabalhar
FIM
